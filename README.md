# 🎬 FilmGuard AI

FilmGuard AI is an AI-powered website piracy detection system that analyzes website content and predicts whether it contains potentially pirated movie-related content.

## 🚀 Features

* Website URL Analysis
* Automatic Content Extraction
* AI-Based Classification
* Piracy Risk Score
* Content Preview
* Modern Web Interface

## 🛠️ Technologies Used

* Python
* Flask
* Scikit-Learn
* Pandas
* BeautifulSoup
* HTML/CSS

## 📊 How It Works

1. User enters a website URL.
2. FilmGuard fetches the webpage content.
3. Text is extracted automatically.
4. AI analyzes the content.
5. Risk score and prediction are displayed.
